# Learning Directory

このディレクトリは、Weaverプロジェクトの学習用タスク管理に使用します。

## 目的

- Rustの非同期処理、所有権/ライフタイム、関数型プログラミングの学習
- 実装を通じた理解の深化
- 学習履歴と進捗の記録

## ファイル管理方式

日付プレフィックス（`YYYY_MM_DD.md`）形式で管理します。

```
dev/learning/
├── README.md           # このファイル
├── 2025_12_28.md       # 2025年12月28日の学習タスク・進捗
├── 2025_12_29.md       # 翌日の学習タスク・進捗
└── ...
```

**重要**: 常に**最新日付のファイル**が現在の学習タスクリストです。
- 過去のファイルは履歴として保存されます
- 新しい日に学習を開始する際は、新しい日付でファイルを作成してください

## ファイルの内容

各日付ファイルには以下を記録します：

### 1. タスクリスト
- [ ] 未完了タスク
- [x] 完了タスク

### 2. 学習メモ
- 実装中に学んだこと
- つまづいた点と解決方法
- 設計上の気づき

### 3. 進捗
- その日に完了したタスク
- 次回の実装予定

## Claude Code への指示

**重要**: このディレクトリ内のタスクは**学習用**です。

- Claude Code は、これらのタスクを**自動実装してはいけません**
- ユーザーが明示的に実装を依頼した場合のみ、サポートしてください
- 質問や実装のヒント、レビューは積極的に行ってください

## 参考ドキュメント

学習タスクは以下のドキュメントに基づいています：

- `dev/docs/requirements/` - 要件定義（最新日付が正）
- `CLAUDE.md` - プロジェクト概要とアーキテクチャ
- `dev/docs/adr/` - アーキテクチャ決定記録

# Learning Directory

このディレクトリは、Weaverプロジェクトの学習用タスク管理に使用します。

## 目的

- Rustの非同期処理、所有権/ライフタイム、関数型プログラミングの学習
- 実装を通じた理解の深化
- 学習履歴と進捗の記録

## ファイル管理方式

**タスク管理**と**実装記録**を分離して管理します。

```
dev/learning/
├── README.md           # このファイル（ディレクトリの使い方）
├── tasks.md            # 全フェーズのタスクリスト（マスター）
├── 2025_12_28.md       # 2025年12月28日の実装記録
├── 2025_12_29.md       # 2025年12月29日の実装記録
└── ...
```

### ファイルの役割

#### `tasks.md` - タスクマスターリスト
- 全 Phase（Phase 1〜8）のタスクを一元管理
- 各タスクの完了状態（✅ ⏳ 🔄）
- Phase ごとのゴールと説明
- **このファイルを見れば、プロジェクト全体の進捗が分かる**

#### `YYYY_MM_DD.md` - 日次実装記録
- その日に実装した内容の詳細
- 学んだこと、つまづいた点、解決方法
- コードスニペットと説明
- **時系列で振り返る際に使用**

### 使い方

**タスクを確認する:**
→ `tasks.md` を開く

**今日の作業を記録する:**
→ `2025_12_29.md`（今日の日付）に実装記録を追加

**過去の実装を振り返る:**
→ 該当日付のファイルを開く

**タスクを完了したら:**
1. `tasks.md` で該当タスクを `[x]` にする
2. 日次ファイルに実装記録を書く
